# Particle-adaptive reconstruction audit (blocker TODO §2, post-processing)

Post-processing audit (no solver rerun) addressing the §5.4 blocker item 3:
**can the particle cloud itself drive a local reconstruction that resolves the
concentrating core better than a uniform global Fourier bandwidth?**

Produced by `experiments/keller_segel/pp_particle_ldg/adaptive_reconstruct.py` from
raw particle clouds saved by `ldg_comparison/simulation.py --save_cloud_snapshots`
(base N=2e4/K=5, refined N=8e4/K=10), reusing the verified hybrid operators in
`experiments/resolution_hybrid/reconstructors.py` (signed residual, raised-cosine
taper, no double-counting).

## Result — honest, MIXED / limited (cf. solver-hybrid plan §9 "limited success")

**Positive.** Global Fourier under-resolves the concentrating core at every feasible
global bandwidth: on the base cloud at `t=2e-4` the global-Fourier peak is only
`2.4k → 78k` for `Kg = 5 → 32` and `S_L2` only `262 → 1293`, while the
particle-adaptive hybrid (low global `Kg=8` + local `Kl=24…40` or a local blob)
reaches peak `≈3.9e5` and `S_L2 ≈ 2180` — i.e. it recovers the core that global
Fourier misses, landing near the deterministic FVM `n=512` anchor (`peak 5.95e5`,
`S_L2 2941`), using a *low* global bandwidth. The reconstruction-free radii
`R_0.5, R_0.8` are unchanged, and the adaptive `S_core` tracks the sharp core
consistent with those radii where global Fourier does not (acceptance criterion 3).

**Caveats (reported, not hidden).**
1. The adaptive reconstructed peak is still sensitive to the *local* bandwidth
   `Kl` (e.g. on the refined cloud at `t=1.2e-4`, peak `545k → 979k` for
   `Kl=24→40`) and overshoots the FVM grid baseline for the finer cloud — so
   adaptive reconstruction trades *global*-bandwidth sensitivity for *local*-
   bandwidth sensitivity; it does not by itself pin a converged peak.
2. The tapered signed residual is only **approximately mass-conserving**:
   `∫u_hyb dx ≈ 25.3` vs `M=10π≈31.4` (~19% deficit), whereas global Fourier
   conserves mass exactly. The local Gaussian blob variant has the same deficit.
3. **Negativity (Gibbs).** Both high-`K` global Fourier *and* the signed hybrid go
   negative; the local blob undershoots least (`u_min ≈ -1.1k` vs `-6.6k` for the
   spectral hybrid at `t=2e-4`, base).

**Conclusion.** Particle-adaptive reconstruction is a real, particle-driven
improvement for the core diagnostics and reduces global-bandwidth sensitivity, but
introduces local-bandwidth sensitivity and signed-residual artifacts (negativity,
~mass deficit). It is reported as a diagnostic improvement, **not** a converged
peak or blow-up estimate. This motivates — but does not yet validate — feeding the
hybrid field into the solver drift (the online solver-level hybrid, plan §4, which
remains to be implemented and verified) and a direct LDG reference (plan §6).

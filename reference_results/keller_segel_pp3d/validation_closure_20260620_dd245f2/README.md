# 3D fully parabolic–parabolic Keller–Segel — validation closure

Closure pass on the remaining validation items for the fully parabolic–parabolic
Keller–Segel particle solver

```
u_t = D_u Δu − χ ∇·(u ∇v),    v_t = D_v Δv + α u − β v,    on T_L^3 (L=12), v0 = 0,
```

chemical generated by the exact decay–injection substep
`μ_v^{n+1} = e^{−βτ} μ_v* + (α/β)(1−e^{−βτ}) μ_u*`. Normalized regime `D=α=β=χ=1`.

- **Base commit:** `dd245f2` (branch `pp3d-validation-closure`, off `main` `6731f15`).
- **Solver / drivers unchanged in dynamics:** the same-cloud `drift_probe` draws no RNG and
  does not enter transport; `test_buffer_equiv.py` confirms `fast==slow`; the K=12 runs here
  reproduce the existing production `R_0.5(T)≈0.149`.
- **No paper file is touched.** Deliverables: verified code+tests, raw data, solver-free
  plotting scripts, figures, and `ANALYSIS_REPORT.md`.
